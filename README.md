## Lancer le projet localement

Cette application requiert [NodeJS](https://nodejs.org/fr).

```bash
npm install -g @angular/cli
git clone https://github.com/Peaulhynes/Peaulhynes.github.io.git
cd Peaulhynes.github.io
npm install
npm start
```

Attendez la fin de la compilation puis allez sur http://localhost:4200.
